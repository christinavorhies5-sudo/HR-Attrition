HR Attrition Diagnostic: Predictive Turnover Modeling
This project identifies core drivers of employee turnover through predictive analytics and data visualization. By analyzing salary benchmarks, professional development opportunities, and tenure, this diagnostic provides actionable strategies aimed at improving workforce retention by 10%.
